## ‚úÖ Summary of the Project

This project analyzes video game datasets from multiple sources to understand **sales performance, user engagement, and market demand**. The analysis combines *vgsales.csv* (sales data) with *games.csv* (user engagement + ratings data). Using data cleaning, transformation, and dashboarding techniques, insights were generated to understand:

* Which games sell the most globally,
* Which genres are most popular,
* Which developers consistently produce high‚Äërated games,
* How wishlist, ratings, and backlog trends indicate user engagement.

## ‚úÖ Business Problem

The gaming industry produces thousands of titles every year, but only a few succeed commercially. Game studios and publishing companies need clear insights into:

* Which **genres and platforms generate higher sales**,
* Whether **higher user engagement (wishlist, plays, ratings)** correlates with sales,
* Which **regions drive maximum revenue**,
* Which **games and studios are consistent top performers**.

Goal: Use data-driven evidence to help publishers and developers make informed decisions on future releases, marketing strategies, and platform targeting.

## ‚úÖ Methodology (3 Steps)

1. **Data Cleaning & Transformation** ‚Äì Handled missing values, merged columns (Title/Name, Genre/Genres), created a unified `Game_Title` field; performed unpivoting for regional sales breakdown.
2. **Exploratory Data Analysis (EDA)** ‚Äì Used Power BI dashboards to analyze ratings distribution, wishlist trends, sales by region, best-selling platforms, and publisher performance.
3. **Insight Generation & Reporting** ‚Äì Built interactive visual dashboards and documented key insights with business implications.

## ‚úÖ Results & Business Recommendations

### üîç Key Insights

* **Action, Adventure, and RPG genres dominate both ratings & wishlist counts**, showing high future revenue potential.
* **EU & North America contribute the highest sales globally**, highlighting ideal marketing regions.
* **Nintendo, Capcom, and EA** have consistently strong performance, indicating brand value drives user purchasing.
* **Sports & Action games generate maximum global sales**, while **Adventure & RPG games generate maximum engagement**.

### üí° Recommendations

* Publishers should **focus on genres with engagement + sales alignment**, especially Adventure & RPG.
* For games with high wishlist but low sales, **consider pricing strategy revisions or seasonal discount campaigns**.
* Platform strategy should prioritize regions: NA/EU ‚Üí PlayStation/Xbox; JP ‚Üí Nintendo.
* Developers should focus on **long-running franchises**, as sequels have built-in audience demand.
